# hubot-j5-example

A hubot script that does the things

See [`src/j5-example.js`](src/j5-example.js) for full documentation.

## Installation

In hubot project repo, run:

`npm install j5-example --save`

Then add **j5-example** to your `external-scripts.json`:

```json
["hubot-j5-example"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
